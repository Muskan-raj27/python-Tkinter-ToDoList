# 📖 Project Description

The To-Do List Application is a desktop GUI application developed using Python and the Tkinter library. It helps users organize their daily tasks by allowing them to add, delete, and manage tasks easily. All tasks are stored in a text file so they remain available even after the application is closed.

---

# 🎯 Objectives

- Build a user-friendly GUI using Tkinter.
- Learn file handling in Python.
- Perform task management operations.
- Save tasks permanently using a text file.
